Design System Reference

Brand &
implementation guide

Typography System
Display / Editorial: Cormorant Garamond — 300 weight for elegance, 500 italic for warmth. Used for hero headlines, section titles, statistics, and pull quotes.
  • Hero H1: 72–96px / weight 300 / tracking −0.01em
  • Section Title: 48–62px / weight 300 / italic accent
  • Card Title: 22–28px / weight 400
  • Stat Numbers: 40–72px / weight 300 / tight tracking
Body System
Interface / Body: DM Sans — 300 weight for refinement, 400 for emphasis. Used for navigation, body copy, labels, forms, and UI elements.
  • Body: 15px / weight 300 / line-height 1.75
  • Small body: 14px / weight 300 / line-height 1.7
  • Labels: 11px / weight 400 / tracking 0.2em / uppercase
  • Nav: 13px / weight 300 / tracking 0.06em
Motion Principles
All animation uses a single custom easing: cubic-bezier(0.16, 1, 0.3, 1) — fast-out, soft-finish. Feels natural and premium.
  • Page load reveals: 1s duration, 20–40px Y offset, staggered
  • Scroll reveals: Intersection Observer, 0.8s, threshold 0.15
  • Hover states: 0.2–0.3s, minimal transform (1–2px Y)
  • Theme transitions: 0.8s full-page, overlay flash prevented
  • Marquee: 25s linear, duplicated content, paused on hover
Conversion Psychology
Every conversion element is designed around executive-level psychology — trust before CTA, scarcity through selectivity, authority through specificity.
  • Gate the inquiry: application form signals exclusivity
  • Specificity over claims: exact metrics, named contexts
  • Restraint communicates confidence: less copy = more authority
  • Criteria list: filters leads, reinforces client ideal profile
  • Multi-step form: investment signals serious intent
Dark/Light Mode Logic
Auto-detection based on local time + prefers-color-scheme fallback. Smooth overlay transition prevents FOUC. Manual override persisted in localStorage.
  • Sunrise/sunset: 6am light mode, 7pm dark mode (approximate)
  • Fallback: prefers-color-scheme media query
  • Manual override: stored in localStorage as 'axiom-theme'
  • Transition: 0.8s body transition on bg/color CSS vars
  • No theme flash: initial theme applied before paint via script
Technical Stack
Recommended implementation for production, optimized for performance, accessibility, and SEO.
  • Framework: Next.js 14 (App Router) + TypeScript
  • Styling: TailwindCSS + CSS custom properties
  • Animation: Framer Motion (scroll) + CSS (hovers)
  • Components: shadcn/ui base, custom design tokens
  • Deploy: Vercel Edge Network, image optimization on
  • Fonts: next/font with display swap, subset loaded
